charles 修改请求和响应

  • 对网站起始进行Focus方便观察
  • 下断点，选择要修改的请求链接下断点
  • charles会对该链接的请求和响应进行拦截

  • Edit Request 修改请求内容

  • Edit Response 修改响应内容

  • 修改完后Execute执行

charles 修改文件

  • 对需要修改的文件选择map local
  • 添加Map to 中的本地替换文件
