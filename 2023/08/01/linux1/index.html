Linux常用命令

介绍使用mac终端操作安卓手机的Linux常用命令，及Linux权限。

Linux常用命令

  • mac安装adb工具
  • brew install –cask android-platform-tools
  • Linux中一切皆文件，命令实际上是系统中的一个二进制文件
  • Linux文件不需要扩展名（加了扩展名也不会改变什么），有时为了区分，才加上扩展名

  1. 根目录 /

  2. 显示当前目录 pwd

  3. cd

    1. cd xxx 切换到指定目录
    2. cd / 切换到根目录
    3. cd ~ 切换到用户目录（安卓里面用户目录就是根目录）
    4. cd .. 返回到上级目录

  4. 相对路径和绝对路径

    1. / 和 ~ 开始到都是绝对路径

    2. ls 显示文件

      1. ls 路径
        1. ls -a 显示全部文件
        2. ls -al 显示详细信息
        3. ls -alh 统计文件大小 （4096代表文件夹）
        4. 一般命令都是小写，选项区分大小写，顺序可颠倒
      2. df -h 显示磁盘分区`

  5. 文件操作

    • 文件类型使用file命令查看（支持通配符file *）文件大小命令du -h *

    • 隐藏文件以.开头

    • 创建文件/文件夹

      • touch 创建空文件
      • mkdir aaa 创建目录aaa
      • mkdir -p aaa/bbb 递归创建目录

    • 删除文件/文件夹

      • rmdir 删除空目录
      • rm xxx 删除文件
      • rm -r 递归删除
      • rm -rf 强制删除

    • 拷贝文件/文件夹

    • 拷贝文件

      • cp 原路径文件 拷贝路径
      • cp a /sdcard
      • cp a b /sdcard
      • cp * /sdcard

    • 拷贝目录

      • cp -r 原路径文件 拷贝路径

    • 移动

      • 移动后改名字（可以不改去掉就行）
      • mv oldname /sdcard/newname
      • mv oldname newname（移动到当前目录下，相当于改名字）

  6. 输出

    1. 输出 echo ：
      1. 输出环境变量 echo $PATH 输出设备名echo $HOSTNAME
      2. 将文本保存在某个文件 echo hinhao! > demo.tex (>输出重定向，>>追加)
    2. 输出文件内容 cat

  7. 搜索

    • grep -i 忽略字母大小写
    • grep -r 递归查找
    • grep -l 搜索到了现实文件名 grep -l xxx * 只列出匹配的文件名
    • grep -irl 文件名 搜索目录
    • find 路径 -name 文件名

  8. 管道命令

    1. 相当于命令的组合使用，把上一个命令的执行结果，作为数据源，交给下一个命令处理
    2. ps -A ｜ grep xxx

  9. 软连接

    1. 符号连接，类似于windows快捷方式，主要存放链接其他文件的路径
    2. ln -s 原文件或目录 软连接

权限

  • ls -l

  • 如图 权限 拥有者 拥有组 文件大小（4096位目录）

  • 权限共有10位

  • 第0位 确定文件类型

    • - 普通文件
    • l 软链接
    • d 目录
    • c 设备文件
    • b 块文件（硬盘/dev）

  • 第1-3位确定所有者有该文件的权限

  • 第4-6位确定所属组者有该文件的权限

  • 第7-9位确定其他用户拥有该文件的权限

    • rwx 可读可写可执行
    • -没有权限
    • r = 4 w = 2 x =1

  • 权限变更

    • chmod u=rwx,g=rx,o=x 文件名
    • chmod o+w 文件名
    • chmod g-x 文件名
    • 一次改变全部 chmod a-x文件名 三种用户都没有x权限
    • chomd 777 xxx

  • 不是所有目录和文件都可以通过这个方法修改，有些需要修改selinux策略
