Docker(一)

基础概念，配置安装，常用命令

基础概念

  1. Docker是一个开源的应用容器引擎，基于Go语言和c语言开发。
  2. 一个轻量级的虚拟机，一键环境部署。
  3. 桌面版’https://hub.docker.com/‘

配置安装

  • 设置国内镜像源
  • settings -> Docker Engine ->添加字典

常用命令

  • 下载镜像 ‘docker pull xxx’

  • 创建容器是基于Docker

  • 编写Dockerfile,打包镜像

    • requirements.txt生成命令 pip freeze > requirements.txt
    • 打包镜像命令docker build -t your_image_name
